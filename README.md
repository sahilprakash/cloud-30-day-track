# 30-Day Cloud & Infrastructure Engineering Track

An intermediate, self-study curriculum covering **networking, AWS, infrastructure automation, and PowerShell** — with a built-in progress tracker (your checkmarks are saved in your browser).

**Live site:** https://sahilprakash.github.io/cloud-30-day-track/

## Structure

| Phase | Days | Focus |
|-------|------|-------|
| 1 — Networking & Scripting Foundations | 1–7 | OSI/TCP-IP, subnetting, DNS/NAT/routing, firewalls, PowerShell |
| 2 — AWS Core Services | 8–14 | IAM, VPC, EC2, S3, ELB/Route 53, RDS/DynamoDB |
| 3 — Infrastructure Automation | 15–21 | IaC, CLI/PowerShell/Bash, CloudFormation, Terraform |
| 4 — Advanced Automation & Ops | 22–28 | Ansible, CI/CD, Docker, CloudWatch, security & cost, Lambda |
| Capstone | 29–30 | Build, monitor, and rebuild a full system from code |

Each day follows: **Objectives → Hands-on Lab → Key Commands → Reference → Checkpoint.**

## Local use

Just open `index.html` in any browser — it's a single self-contained file, no build step.
